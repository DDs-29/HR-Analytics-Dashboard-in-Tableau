HR Analytics Dashboard
This interactive HR Analytics Dashboard was built using Tableau to visualize key HR metrics such as employee count, attrition rate, job satisfaction, and demographic analysis. The dashboard provides insights into department-wise attrition, age-wise employee distribution, and education field trends, helping HR teams make data-driven decisions.

Key Features:

Employee attrition analysis by department, gender, and age group
Job satisfaction ratings across different job roles
Education field-wise attrition trends
Interactive visualizations for better decision-making
